📌 Overview
A web-based automation tool designed to bridge the gap between complex AI models and educational planning. This application allows educators and students to generate structured, professional course outlines—including objectives, weekly modules, and assessments—in seconds.

🚀 Key Features
• AI-Powered Generation: Leverages the OpenRouter API to access state-of-the-art LLMs for high-quality content.

• Prompt Engineering: Uses optimized System and User roles to ensure the output is pedagogically sound and professionally structured.

• Modern Web Stack: Built using HTML, CSS, and JavaScript for a lightweight, zero-dependency user experience.

• Responsive Design: Fully functional across desktops and mobile devices using CSS Flexbox.

• One-Click Utility: Includes a "Copy to Clipboard" feature for immediate use in documents or LMS platforms.

🛠️ Technical Challenges Overcome
• Asynchronous API Handling: Implemented JavaScript Promises and Async/Await to manage API latency, ensuring the UI remains responsive during data fetching.

• UX Optimization: Integrated a dynamic loading state to provide real-time feedback to users during the generation process.

• Cross-Platform Portability: Migrated from a desktop-based Python/Tkinter architecture to a Web-stack to enable access for 100+ learners without local installations.

📊 Real-World Impact
• Productivity: Reduces manual curriculum planning time by approximately 60%.

• Standardization: Ensures a consistent format for educational content across different subjects.
